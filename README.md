# プロポフォール–レミフェンタニル 相乗効果ナビ

プロポフォールとレミフェンタニルの **相乗効果（応答曲面 / Greco モデル）** と、**投与量 → 効果部位濃度（Ce）→ 反応確率の時間予測** を可視化する麻酔教育ツール。2つのタブで構成されます。

**① 応答曲面（マニュアル）**
効果部位濃度（Ce）の動作点を曲面上のクリック／ドラッグ、スライダー、プリセットで設定でき、4つのエンドポイント（意識消失・喉頭鏡/挿管・電気テタヌス・圧痛）について「その刺激に無反応の確率」をリアルタイム表示します。

**② 投与シミュレータ（PK）**
年齢・体重・身長・性別と PKモデル（プロポ：Schnider既定 / Marsh / Eleveld 2018、レミ：Minto既定 / Eleveld 2017）を選び、ボーラス／持続を入力すると、3-コンパートメント＋効果部位モデル（RK4）で **Ce の時間推移** を計算。応答曲面上に **Ce 軌跡** を描き、選んだ刺激の「目標無反応％」に達する **予測時間（逆算）** を表示します。時刻スクラブで任意時点の Ce・確率を確認できます（ショット導入／持続導入の比較に）。

- **完全オフライン対応**（Service Worker でキャッシュ）。一度開けば電波がなくても動作。
- **iPhone / iPad 対応**。Safari で開き「ホーム画面に追加」すると全画面アプリとして使えます。
- 単一 HTML（外部依存なし）。

## 使い方（iPhone / iPad）

## モデル
**PD（応答曲面）**：効果 = Uⁿ/(Uⁿ+1)、U = C_P/C50_P + C_R/C50_R + α·(C_P/C50_P)(C_R/C50_R)。「相加（相乗なし）」は同じ式で α=0 とした理論値で、その差が相乗による上乗せ。パラメータ：Kern SE, et al. *Anesthesiology* 2004;100:1373 ／ Johnson KB, et al. *Anesth Analg* 2008;106:471（検証）。モデル形：Greco WR *Pharmacol Rev* 1995；Minto CF *Anesthesiology* 2000。

**PK（投与シミュレータ）**：3-コンパートメント＋効果部位（dCe/dt = k_e0(C_p − Ce)）を RK4 で積分。Schnider（*Anesthesiology* 1998/99, k_e0 0.456, LBM=James）／Marsh（*BJA* 1991, modified k_e0 1.2）／Eleveld propofol（*BJA* 2018, Al-Sallami FFM, オピオイド併用前提）／Minto（*Anesthesiology* 1997, LBM=James）／Eleveld remifentanil（*Anesthesiology* 2017, Al-Sallami FFM）。各式は原著＋独立実装で照合済み。

## 注意
母集団 PD/PK モデルに基づく確率・濃度の推定です。Ce は選んだ PK モデルに依存し、C50 は元論文が用いたモデルで較正されているため絶対値の整合は保証されません。プリセット・予測時間は概念理解のための例で、投与量や導入時間の推奨ではありません。Schnider/Minto は高度肥満で LBM(James式) が破綻しえます。臨床的観察と併せて判断してください。
