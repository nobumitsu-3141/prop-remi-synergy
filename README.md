# プロポフォール–レミフェンタニル 相乗効果ナビ

プロポフォールとレミフェンタニルの **相乗効果（応答曲面 / Greco モデル）** を可視化する麻酔教育ツール。
効果部位濃度（Ce）の動作点を曲面上のクリック／ドラッグ、スライダー、プリセットで設定でき、4つのエンドポイント（意識消失・喉頭鏡/挿管・電気テタヌス・圧痛）について「その刺激に無反応の確率」をリアルタイム表示します。

- **完全オフライン対応**（Service Worker でキャッシュ）。一度開けば電波がなくても動作。
- **iPhone / iPad 対応**。Safari で開き「ホーム画面に追加」すると全画面アプリとして使えます。
- 単一 HTML（外部依存なし）。

## 使い方（iPhone / iPad）
1. Safari で公開 URL を開く
2. 共有ボタン → **ホーム画面に追加**
3. アイコンから起動（オフラインで動作）

## モデル
効果 = Uⁿ/(Uⁿ+1)、U = C_P/C50_P + C_R/C50_R + α·(C_P/C50_P)(C_R/C50_R)。
「相加（相乗なし）」は同じ式で α=0 とした理論値で、その差が相乗による上乗せ。
パラメータ：Kern SE, et al. *Anesthesiology* 2004;100:1373 ／ Johnson KB, et al. *Anesth Analg* 2008;106:471（検証）。モデル形：Greco WR *Pharmacol Rev* 1995；Minto CF *Anesthesiology* 2000。

## 注意
母集団薬力学モデルに基づく確率推定であり、個々の患者・併用薬・侵襲度の差を保証しません。プリセットは概念理解のための例で、投与量の推奨ではありません。臨床的観察と併せて判断してください。
